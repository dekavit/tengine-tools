# tengine-tools

## Usage

Set tengine library path

`tengine-tools set <tengine-lib-path>`

Cleate tengine project

`tengine-tools init <project-name>`

## Installation

```
make
make install
```

## What is Tengine?

Tengine is a game engine written in the D programming language.

https://github.com/KCCTdensan/tengine
